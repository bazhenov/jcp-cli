## Developing

We use file-backend for storing secrets instead of platform keychain when running locally. macOS prompts for a password on each recompile, slowing development.

## Code Review

1. Propose contributions as pull requests (PRs) in this repo (https://github.com/jetbrains/jcp-cli).
2. Assignees or authors merge PRs.
3. GitHub Review conversations are resolved by the initiator, confirming the issue was addressed. Nitpicks can be added as ad-hoc comments.
