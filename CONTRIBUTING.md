# Code Review

1. When you have a contribution, please propose it as a pull request (PR) in this repo (https://github.com/jetbrains/jcp-cli).
2. The person who created the PR will merge it.
3. GitHub Review conversations are resolved by the person who started the conversation (not the author of PR). This way author of the review is confirming the the issue was addressed properly. Nitpicks can be added as ad-hoc comments that do not require resolution.
